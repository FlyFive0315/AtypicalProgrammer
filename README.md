# 非典型程序员助手

非典型程序员助手，帮你做一些程序员不太喜欢的工作。

作为一个程序员，经常会被要求承担一些非开发工作，这些工作不做不行，做了又感觉比较枯燥重复，好像在浪费生命，既然代码助手已经很常见了，来一个非代码的助手不过分吧。
